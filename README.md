# Camtasy

A fancy dancy cam server based on the gorgeous work of that lovely
[tenderlove](http://tenderlovemaking.com/2014/03/26/webcam-photos-with-ruby.html)

## Installation

    $ gem install camtasy

## Usage

    $ camtasy server
    $ camtasy client | open -f -a /Applications/Preview.app
